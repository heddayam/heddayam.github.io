%% Research
\section*{Research}
\vspace{4pt}
\noindent{\large\textbf{PhD Student, University of Chicago CS}} \hfill 2021--2026
\begin{cvitems}
    \item Demonstrated that chain-of-thought internalization fails on complex reasoning [1]. Proposed post-think distillation, a new compression target that preserves the teacher's solution path and achieves the best accuracy--efficiency trade-off on competition math. Designed evaluation pipelines to diagnose failure modes across curriculum stages and distillation targets.
    \item {[ongoing]} Built and deployed \href{https://civicchats.org}{CivicChats.org}, an end-to-end AI platform for civic deliberation. Built CivicEval, an automated evaluation pipeline whose outputs inform iterative prompt optimization over collected user data. Running a controlled experiment on voter understanding. Follow-up work on large-scale data curation, synthetic data generation, and preference-based training to elicit coherent model behavior beyond surface style. \href{https://open.substack.com/pub/cichicago/p/civicchats-building-ai-to-support}{[blog]}
    \item Built a 25.6K Supreme Court opinion--summary benchmark [2] with an evaluation framework comparing automatic metrics, LLM-as-judge, and expert human review; showed fine-tuning improves automatic metrics while increasing hallucination rate. Neither standard metrics nor LLM-as-judge reliably catch subtle failures in frontier model outputs.
    \item Developed a method for at-scale automatic extraction of causal narratives and applied it to 4.2M sentences from a century of U.S.\ inflation coverage [4]; found causal narratives predict consumer expectations and spread via contagion and differentiation.
    \item Collected $\sim$400 human bargaining interactions with real incentives [3]; language reduces price variance 3$\times$ and simple linguistic features outperform transformers at predicting outcomes.
\end{cvitems}
